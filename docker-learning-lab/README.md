# Docker Learning Lab

This repository documents my hands-on practice while learning Docker.

Each day focuses on a specific Docker concept including containers,
volumes, custom images, and container-host interaction.

Topics Covered

Day 1 – Docker Basics & Images  
Day 2 – Container Interaction  
Day 3 – Docker Volumes  
Day 4 – Custom Nginx Docker Image  
Day 5 – Popular Docker Images & Practice  
Day 6 – Maven with Docker (Basic)  
Day 7 – Docker Compose for Maven  
Day 8 – Maven Wrapper & Docker Integration  
Day 9 – CI/CD with GitHub Actions  
Day 10 – Multi-Container Apps (WordPress + MySQL)  
Day 11 – GitHub Actions with Docker (Python/Flask)  
Day 12 – Maven & Docker Integration Deep Dive  
Day 13 – Advanced Java CI/CD with Artifacts

Goal:
Understand how containers work and how to manage them in real-world DevOps environments.
